## Компиляция проекта ##

### Gulp ###

```
$ npm install -g gulp
```

### Установка зависимостей ###

```
$ npm install
```

### Сборка + сервер ###

Проект соберётся в папку /build и запустится локальный сервер с синхронизацией.

```
$ gulp
```
### Сборка в папку build ###

Проект соберётся в папку /build

```
$ gulp build
```

### Удаление папки build ###

```
$ gulp clean
```
